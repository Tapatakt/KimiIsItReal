# Kimi Is It Real — расширение для Brave/Chrome

***Дисклеймер: Проект навайбкожен при помощи Kimi Code.***

## Что делает

1. Выделяешь текст на любой странице.
2. Нажимаешь **Ctrl+Shift+K** (на Mac: **Cmd+Shift+K**).
3. Открывается новая вкладка [kimi.com](https://kimi.com) с уже вставленным промптом и автоматически отправляется.

Промпт состоит из трёх частей:
- **Префикс** (настраивается)
- **Выделенный текст**
- **Суффикс** (настраивается)

По умолчанию получается:

> Аккуратно проверь корректность фактов в этом тексте:
> 
> "<выделенный текст>"
> 
> Не удовлетворяйся одним источником на факт, ищи альтернативные. Приводи ссылки. В конце вынеси краткий вердикт по достоверности информации.

## Известные проблемы

**Первое использование сразу после перезагрузки расширения** иногда открывает пустую вкладку Kimi без промпта. Со второго раза и далее всё работает стабильно. Это связано с особенностями загрузки content script в Manifest V3 после reload.

## Установка

1. Скачай и распакуй релиз.
1. Открой в Brave (или Chrome) → `brave://extensions/` (или `chrome://extensions/`).
2. Включи режим разработчика (переключатель в правом верхнем углу).
3. Нажми **Load unpacked** / **Загрузить распакованное расширение**.
4. Выбери папку `extension` из распакованного проекта.

## Настройка промпта

1. Нажми на иконку расширения в панели инструментов.
2. Измени текст "До выделенного" и/или "После выделенного".
3. Нажми **Сохранить**.

## Горячая клавиша

По умолчанию: `Ctrl+Shift+K`.  
Её можно поменять в настройках расширений (`brave://extensions/shortcuts`).

## Структура

- `manifest.json` — конфигурация Manifest V3.
- `background.js` — service worker, слушает горячие клавиши.
- `content.js` — получает выделенный текст на странице.
- `kimi.js` — вставляет промпт в поле ввода Kimi.
- `popup.html`, `popup.js` — настройка промпта.
- `icons/` — иконки расширения.
